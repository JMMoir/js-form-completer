How to use the form completer

Drag the Form Completer box to you bookmarks.

Once on a page with radio buttons, checkboxes or drop down lists click the bookmark.

Feature requests and bugs

If you would like any improvements to the code or bugs(<--- this is very subjective since the tool is very rough and ready) to be addressed
please create an issue here - https://github.com/JMMoir/js-form-completer/issues, I'll get round to it when I have time.
PRs welcome.
